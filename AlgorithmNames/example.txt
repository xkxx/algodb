<li> <a href="/wiki/Backtracking" title="Backtracking">Backtracking</a>: abandon partial solutions when they are found not to satisfy a complete solution</li>
<li> <a href="/wiki/Beam_search" title="Beam search">Beam search</a>: is a heuristic search algorithm that is an optimization of <a href="/wiki/Best-first_search" title="Best-first search">best-first search</a> that reduces its memory requirement</li>
<li> <a href="/wiki/Beam_stack_search" title="Beam stack search">Beam stack search</a>: integrates backtracking with <a href="/wiki/Beam_search" title="Beam search">beam search</a></li>
<li> <a href="/wiki/Best-first_search" title="Best-first search">Best-first search</a>: traverses a graph in the order of likely importance using a <a href="/wiki/Priority_queue" title="Priority queue">priority queue</a></li>
<li> <a href="/wiki/Bidirectional_search" title="Bidirectional search">Bidirectional search</a>: find the shortest path from an initial vertex to a goal vertex in a directed graph</li>
<li> <a href="/wiki/Bloom_filter" title="Bloom filter">Bloom filter</a>: a constant time and memory check to see whether a given element exists in a set. May return a false positive, but never a false negative.</li>
<li> <a href="/wiki/Breadth-first_search" title="Breadth-first search">Breadth-first search</a>: traverses a graph level by level</li>
<li> <a href="/wiki/D*" title="D*">D*</a>: an <a href="/wiki/Incremental_heuristic_search" title="Incremental heuristic search">incremental heuristic search</a> algorithm</li>
<li> <a href="/wiki/Depth-first_search" title="Depth-first search">Depth-first search</a>: traverses a graph branch by branch</li>
<li> <a href="/wiki/Dijkstra%27s_algorithm" title="Dijkstra's algorithm">Dijkstra's algorithm</a>: A special case of A* for which no heuristic function is used</li>
<li> <a href="/wiki/General_Problem_Solver" title="General Problem Solver">General Problem Solver</a>: a seminal theorem-proving algorithm intended to work as a universal problem solver machine.</li>
<li> <a href="/wiki/Iterative_deepening_depth-first_search" title="Iterative deepening depth-first search">Iterative deepening depth-first search</a> (IDDFS): a state space search strategy</li>
<li> <a href="/wiki/Jump_point_search" title="Jump point search">Jump point search</a>: An optimization to A* which may reduce computation time by an order of magnitude using further heuristics.</li>
<li> <a href="/wiki/Lexicographic_breadth-first_search" title="Lexicographic breadth-first search">Lexicographic breadth-first search</a> (also known as Lex-BFS): a linear time algorithm for ordering the vertices of a graph</li>
<li> <a href="/wiki/Uniform-cost_search" title="Uniform-cost search" class="mw-redirect">Uniform-cost search</a>: a <a href="/wiki/Tree_traversal" title="Tree traversal">tree search</a> that finds the lowest cost route where costs vary</li>
<li> <a href="/wiki/SSS*" title="SSS*">SSS*</a>: state space search traversing a game tree in a best-first fashion similar to that of the A* search algorithm</li></ul>
